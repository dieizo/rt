: Essa linha informa ao navegador que o documento está em HTML5. : Esta tag é o elemento raiz que envolve todo o conteúdo da página. : Nesta seção, você pode incluir informações sobre o documento, como o título da página e links para estilos. : Define o título da página, que aparece na aba do navegador. <body>: Aqui é onde o conteúdo visível da página é colocado, como textos, imagens e links. Principais tags HTML <h1> a <h6>: Usadas para criar cabeçalhos. <h1> é o mais importante, enquanto <h6> é o menos. <p>: Define um parágrafo de texto. <a>: Cria um link. Por exemplo: <a href="https://www.exemplo.com">Clique aqui</a>. <img>: Insere uma imagem. Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">. <ul> e <ol>: Criam listas não ordenadas (com marcadores) e ordenadas (numeradas) <li>: Usada dentro de listas para definir os itens. <div>: Um contêiner genérico para agrupar elementos e aplicar estilos. <span>: Usada para agrupar partes do texto dentro de um parágrafo ou outro elemento, normalmente para aplicar estilos.
